# Lamination Peel Comparator

보호필름이 양면에 부착된 패널에서 하면 필름을 XYZ 궤적으로 당길 때, 상면 필름의 역박리 위험을 조건 A/B 사이에서 비교하는 Windows 데스크톱 애플리케이션입니다.

> [!IMPORTANT]
> 이 프로그램은 보정 전 reduced-order 비교 모델입니다. 계산값은 실제 불량률, 합격 판정 또는 안전 보증값이 아닙니다. 동일한 가정 아래 궤적과 공정 조건의 상대 우열을 선별하는 용도로만 사용하십시오.

## cohesive-v6-visual-loads 계산 모델

현재 계산 코어는 궤적 진행률과 실제 박리 상태를 분리합니다.

1. 각 waypoint의 Speed를 해당 점의 목표속도로 해석하고, 두 점 사이를 일정 가속도(속도의 시간 선형 보간)로 보간합니다.
2. 하면 계면의 각 노드에서 Kendall 계열 에너지 방출률을 계산합니다.
3. 그립의 누적 면내 이동으로 지급된 필름 길이 안에서만 연결 손상 전선이 접근할 수 있습니다. 수직 Z 이동은 박리각과 힘을 바꾸지만 계면 도달 면적을 공짜로 넓히지 않습니다.
4. G_bottom >= Gamma_bottom이고 현재 손상 전선과 연결되며, 방향성 증분 일 max(F·dr, 0)이 필요한 파괴 일을 공급한 노드만 손상시킵니다.
5. 실제 하면 손상장의 외곽에서 박리 전선을 구하고, 현재 전선 중심–그립 거리로 풀테이프 장력을 계산해 반력 F=(Fx,Fy,Fz)에 적용합니다.
6. 그립과 전선의 레버 암으로 M=(r_grip-r_front) x F를 구하고, 면내 힘의 패널 하면 편심 모멘트도 더합니다.
7. 수직력과 등가 굽힘·비틀림 하중을 상면 PSA 위의 탄성 패널에 적용합니다.
8. 상면 cohesive damage가 생기면 노드별 foundation 강성을 낮추고, 변위-손상-강성 행렬을 반복 갱신해 수렴시킵니다.

trajectory_progress는 waypoint 순서와 현재 국부 구간만으로 정한 재생용 진단값일 뿐이며 박리율로 사용되지 않습니다. P1→P6 최종 벡터 투영이나 최종 경로 길이로 정규화하지 않으므로 P6를 바꾸어도 P1~P4의 시간·위치·속도·물리 상태는 바뀌지 않습니다. 실제 하면 박리율은 bottom_peel_ratio에 저장된 면적 가중 cohesive damage입니다. 순수 수직 경로가 끝났다는 이유만으로 전체 하면이 100% 박리되지 않습니다.

### 절대 Z와 P1 초기조건

- 패널 표면을 Z=0으로 두며 모든 Z 입력은 절대 좌표입니다. 조건별 P1 Z를 0으로 다시 맞추지 않습니다.
- 기본 초기조건은 풀테이프가 패널 코너에 부착되고 헤드가 이미 P1을 유지하는 정적 평형입니다. P1 이전 속도·상승거리·파괴 일은 추정하지 않습니다.
- 따라서 P1의 경로 증분은 0이라 하면 손상이 자동 진전하지 않지만, 실제 박리 전선–그립 벡터로 초기 박리각, Fx/Fy/Fz, 모멘트와 패널 들림은 계산됩니다.
- P1 Speed=0은 정상적인 정지 출발입니다. P1→P2에서는 0에서 P2 목표속도까지 가속합니다. 움직이는 한 구간의 양 끝 속도가 모두 0인 경우만 입력 오류입니다.
- 속도 0에서 속도 의존 점착에 0의 거듭제곱 문제가 생기지 않도록 `quasi_static_speed_mm_s`(기본 0.1 mm/s)를 파괴에너지 환산의 하한으로 사용합니다. 이것은 실제 정지 마찰이나 dwell 시험을 대체하지 않는 보정 가정입니다.

P1 이전의 계측 궤적이 있다면 각 조건의 선택 필드 `initial_approach`에 2개 이상의 waypoint를 넣을 수 있습니다. 마지막 점은 본 궤적 P1의 XYZ와 목표속도에 정확히 연결되어야 합니다. 이 경로는 테이프가 부착된 pre-history로 해석되고 결과 시간축에서 P1 이전은 음수, P1은 0초로 기록됩니다. 옵션이 없으면 `initial_state_mode=p1_equilibrium`, 있으면 `specified_approach`입니다. 앱은 프로젝트 JSON을 열고 다시 저장할 때 이 필드를 보존합니다.

### 풀테이프 장력 법칙과 민감도

`max_pull_force_n`을 모든 프레임의 상시 하중으로 사용하지 않습니다. 각 조건의 P1에서 전선 중심–그립 거리 `L_p1`을 기준으로 다음 단방향 스프링 법칙을 사용합니다.

```text
tension = max(0, initial_preload + tape_stiffness × (L - L_p1))
tension = min(tension, max_pull_force)
```

장력이 0이면 반력, 상면 하중과 패널 들림도 0이며 압축력은 만들지 않습니다. `max_pull_force_n`은 장비가 공급할 수 있는 상한일 뿐입니다. 기본 실행은 초기장력 0.0 / 0.5 / 1.5 N의 3개 조합만 A/B에 paired 방식으로 적용합니다. 풀테이프 등가 인장강성은 일반 PET 필름 자료로 추정한 2.25 N/mm을 고정 사용하며, 강성 자체는 더 이상 민감도 축으로 반복하지 않습니다.

- **Equal preload(기본):** A/B의 P1 장력을 같게 두고 각 조건의 `L_p1`을 별도 기준으로 사용합니다. 절대 Z가 만드는 박리각 차이는 유지하면서 미지의 초기장력 차이는 제거합니다.
- **Shared rest length(선택):** 동일한 자연 길이와 강성을 가정합니다. A/B의 P1 거리 차이가 초기장력 차이로 이어집니다. 자연 길이를 직접 입력하지 않고 A/B 기준 길이를 대신 쓰면 참고 민감도로만 표시하고 확정 판정에 사용하지 않습니다.

결과의 `풀테이프 장력 민감도` 탭은 초기장력 3개 행과 고정 강성 1개 열의 판정 표, 실제 단위값, 셀별 gate·위험·손상·들림·비틀림·힘·장력, Peak Rtop 및 하면 박리율 변화를 보여줍니다. CSV와 HTML에도 모든 초기장력 조합을 저장합니다.

### 풀테이프 강성의 고정 추정값

v0.5.8부터 풀테이프 강성은 장력 sweep에서 제외하고 `tape_stiffness_n_per_mm = 2.25 N/mm`으로 고정합니다. 이 값은 일반적인 PET 풀테이프의 등가값을 다음처럼 환산한 reduced-order 가정입니다.

```text
PET Young's modulus       E = 4.50 GPa
PET backing thickness     t = 50 µm
pull-tape width            w = 10 mm
effective compliant path  L = 1,000 mm
k = E × (w × t) / L       = 2.25 N/mm ≈ 2.25 N/mm
```

참고한 공개 자료는 Mitsubishi Polyester Film의 Hostaphan PET 일반 데이터(Young's modulus MD 약 4.50 GPa)와 3M의 PET 테이프 제품 자료(통상 40–82 µm급 PET 테이프)입니다. 실제 장비의 풀테이프 폭·두께·유효 자유길이·접착층 변형을 직접 계측하지 않았으므로 2.25 N/mm은 재료 식별값이 아니라 현재 reduced-order 모델에 넣는 고정 추정값입니다. 초기장력만 0.0 / 0.5 / 1.5 N으로 남겨 미지 장력 조건을 비교합니다.

자료 링크: <https://www.m-petfilm.de/wp-content/uploads/General-Data-Sheet-7_24_2025.pdf>, <https://multimedia.3m.com/mws/media/2366325O/3m-polyester-protective-tape-336.pdf>, <https://www.3m.com/3M/en_US/p/dc/v000368151/>

### 제공된 A/B 데이터

[`examples/measured_ab.peel.json`](examples/measured_ab.peel.json)을 앱에서 바로 열 수 있습니다. 조건 A의 P1 Z=4 mm, 조건 B의 P1 Z=14 mm와 P1 Speed=0이 그대로 저장되어 있습니다. B의 P1~P5 Z는 A보다 모두 10 mm 높고 상대 Z 증분은 같으며, P5→P6의 XY와 waypoint 목표속도는 조건별 값이 유지됩니다.

## 입력과 공통 가정

조건 A/B의 측정 입력:

- 패널 폭·높이·두께·코너 반경
- 상·하면 PET 두께, PSA 두께, 점착력
- 풀테이프 시작 코너, 폭, 길이
- 정확히 6개인 절대 XYZ·waypoint 목표속도 궤적
- 선택적인 P1 이전 `initial_approach` 궤적

두 조건에 동일하게 적용되는 주요 AssumptionSet 값:

- 점착력 시험 폭·각도
- 패널·PET 탄성률과 포아송비
- PSA 탄성률
- 최대 인장력
- 혼합모드/테이프 신장 에너지 계수
- 필름 지급 길이(그립의 누적 면내 이동)에 따른 등방성 전선 도달 범위
- 손상 후 최소 foundation 강성
- 손상 수렴 허용오차와 최대 반복 횟수
- 하면 완료 gate(기본 98%)
- 정지점의 속도 의존 점착 환산에 쓰는 준정적 속도 하한
- 메쉬 크기와 시간 단계 수
- 불확실성 표본 수와 robust 판정 최소 표본 수

gf는 1 gf = 0.00980665 N으로 변환합니다. 실제 시험 폭·각도·속도를 모르면 파괴에너지 환산값도 불확실하므로, 가능한 경우 시험 조건과 로드셀 데이터를 이용해 보정해야 합니다.

## A/B 판정

판정은 다음 순서입니다.

1. 최종 하면 박리율이 98% 미만인 조건은 완료 gate에서 탈락합니다.
2. gate를 통과한 조건끼리 아래 지표를 낮을수록 좋은 방향으로 비교합니다.
   - Peak Rtop
   - Rtop >= 1 최대 면적
   - 임계 초과 지속시간
   - 상면 최종 cohesive damage 면적
   - 최대 패널 들림
   - 최대 비틀림
3. 한 조건이 나머지 지표에서 나빠지지 않으면서 적어도 하나에서 유의하게 좋을 때만 Pareto 우세로 판정합니다.
4. 지표별 우열이 교차하면 inconclusive로 보류합니다.
5. 3개 초기장력 조합 전체를 분모로 A/B 우세율을 계산합니다. 동률과 판정 보류를 분모에서 빼지 않습니다.
6. 한쪽 우세율이 80% 이상이고 반대쪽 우세가 10% 이하이면 robust, 한쪽이 60% 이상이면 weak, 그 외에는 inconclusive입니다.
7. 재료 불확실성을 선택했는데 표본이 `minimum_robust_samples`보다 적으면 robust_a 또는 robust_b 판정을 허용하지 않습니다.

`tension_sweep`은 고정된 풀테이프 강성에서 초기장력만 바꾸는 구조적 민감도이고, `material_uncertainty`는 PSA·패널 강성·시험 폭·각도 등의 Latin Hypercube 불확실성입니다. 기본 실행은 초기장력 3조합 × A/B = 6회만 계산합니다. 재료 불확실성을 켜면 선택 초기장력 조합에 적용하며, 고급 설정에서 중첩하면 기본 3 × 24 × 2 = 144회가 됩니다. UI에 예상 해석 횟수를 미리 표시합니다.

이전 프로젝트 호환을 위해 `vertical_front_reach_factor` 필드는 읽고 저장하지만 v5 계산에서는 사용하지 않습니다. 이 값을 바꿔도 박리 결과는 변하지 않습니다.

CSV에는 시간별 그립 X/Y/Z, 실제 보간 속도, 박리각, Fx/Fy/Fz와 합력, 모멘트, 실제 장력과 전선–그립 거리, 박리율, 상면 위험도·손상 면적, 패널 들림·비틀림, 상면 PSA 정상반력 합력과 반력 중심이 포함됩니다. 앱 결과 탭에서도 속도·박리각, 세 힘 성분과 상면 PSA 정상반력 이력을 확인할 수 있습니다. HTML 보고서와 전체 결과 JSON에는 `model_version=cohesive-v6-visual-loads`, `calibration_version`, 장력 법칙과 조합, 초기상태 모드, main waypoint 프레임, gate 결과, Pareto 판정 근거와 수렴 진단이 기록됩니다.

## 시각화의 의미

v0.5.3부터는 이전의 고정 카메라 표현을 직교 orbit camera로 대체했습니다. 상면·쿼터·정면·측면 버튼으로 정확한 시점을 선택하거나 A/B 애니메이션 중 하나를 좌클릭 드래그해 yaw/elevation을 직접 조절할 수 있습니다. 두 카메라는 항상 paired 상태로 동기화되며, 더블클릭하면 쿼터뷰로 복귀합니다. 정밀 비교를 위해 카메라는 드래그 중 포인터를 즉시 따르고 버튼을 놓으면 관성 없이 멈춥니다. 상면뷰에서는 Z 축이 보는 방향과 일치해 면외 편이가 숨고, 낮은 elevation에서는 필름 들림이 크게 보입니다. 모든 각도는 동일한 균일 스케일의 affine 직교 투영을 사용하므로, 뷰를 회전해도 프레임마다 화면 맞춤으로 피사체가 늘어나거나 찌그러지지 않습니다.

앱의 동기화 애니메이션은 `bottom_damage_frames`를 공유 꼭짓점 기반의 연속 필름 표면으로 만들고, 실제 시간별 그립 XYZ와 함께 현재 orbit camera에 투영합니다. 계면 손상은 필름 재료를 삭제하거나 독립 타일로 분할하지 않고 각 표면 꼭짓점의 연속적인 들림만 제어하므로, 부분 박리 중에도 필름은 하나의 닫힌 직각 사각 외곽과 연결된 메시를 유지합니다. 미박리 고립 주머니는 시각화에서 연속 표면으로 브리지하되 표시 damage 합계를 보존하며, 밝은 박리 전선은 동일한 표시장의 `damage=0.5` 등고선입니다. 원본 손상값·박리율·힘·위험도 계산은 변경하지 않습니다.

v0.6.0부터 상면 필름은 패널 색상 위의 위험도 얼룩이 아니라 독립된 청록색 전체 시트로 렌더링됩니다. `panel_z_frames_mm`, `top_damage_frames`, `top_risk_frames`를 같은 메시에서 결합해 쿼터·정면·측면에서도 필름 경계와 상면 damage=0.5 전선을 확인할 수 있습니다. 접착된 상면 필름은 패널 변위를 따르고 완전 손상부는 기준면에 남기는 표시 가정을 사용합니다. 실제 상면 필름의 박리 후 처짐·주름은 별도 멤브레인 해석이 없으므로 계산하지 않으며, 작은 패널 들림은 결과 전체의 고정 배율로 확대해 표시합니다. 배율은 프레임이나 카메라 각도에 따라 자동 재조정되지 않습니다.

풀테이프는 실제 파지점과 하면 필름 모서리를 잇는 폭 있는 리본으로, 장비는 조(jaw)와 헤드 본체로 표시합니다. 화면의 노란 화살표는 해석된 장비 하중 Fx/Fy/Fz의 현재 방향과 상대 크기이며, 청록 화살표는 손상 후 유효 foundation 강성으로 계산한 상면 PSA 정상반력입니다. 현재 Winkler foundation은 상면 계면 전단장을 해석하지 않으므로 청록 화살표를 3축 계면 traction으로 해석하면 안 됩니다. `상면 필름`, `하면 필름`, `헤드/테이프`, `힘 벡터` 버튼으로 레이어를 A/B에서 동시에 켜고 끌 수 있습니다.

v0.5.4의 형상 계층은 파란 직각 사각 외곽을 현재 공정의 트리밍 전 적층물·보호 필름·damage 메시 영역으로 사용합니다. 그 안의 주황 라운드 사각형은 향후 완성 셀 트림 라인이고, 녹색 패드는 180° 회전된 셀의 상단 직선 구간 전체에 접합니다. 하단의 검정 아일랜드 커버와 분홍 필·원 홀 라인은 향후 내부 트림 참조이며 현재 cohesive 계산 영역을 잘라내지 않습니다. 기본 값은 외곽 여유 1.5 mm, 완성 셀 코너 R 6 mm, 패드 높이 3 mm이며 조건 편집기에서 변경할 수 있습니다.

- 박리 애니메이션의 인접 셀이 꼭짓점을 공유해 필름 연속성과 전체 외곽을 보존
- 애니메이션 박리 전선이 위상 보정된 하면 손상장의 직선·파형 경계를 추종

## 개발 환경

Python 3.11 이상과 Windows PowerShell을 권장합니다.

    python -m venv .venv
    .\.venv\Scripts\Activate.ps1
    python -m pip install --upgrade pip
    python -m pip install -e ".[dev]"
    python -m lamination_sim

테스트와 패키지용 자체 점검:

    python -m pytest
    python -m lamination_sim --self-test

    powershell -ExecutionPolicy Bypass -File .\scripts\audit_dist.ps1

## 남은 한계

- 박리 필름을 완전한 비선형 셸·접촉·대변형 문제로 풀지 않습니다.
- 상면 필름의 박리 후 자유 처짐·좌굴·주름과 상면 계면 전단 traction은 해석하지 않습니다. 애니메이션의 상면 분리 높이는 정상변위와 damage를 결합한 상태 표시이며 실제 자유 필름 형상 예측이 아닙니다.
- waypoint 사이 보간은 일정 가속도이며 jerk 제한 S-curve, 축별 가속도 한계와 제어기 추종오차를 아직 모델링하지 않습니다.
- 기본 P1 정적 평형은 미지의 접근 이력, dwell, 관성 및 잔류 preload를 0으로 둡니다. 영향이 크면 계측한 `initial_approach`와 로드셀 이력을 사용해야 합니다.
- 절대 Z는 평탄한 패널 기준면 Z=0을 가정합니다. 제공 실측 preset의 패널 두께는 56 µm(0.056 mm)입니다. v5의 필름 지급 제약은 누적 면내 그립 이동과 코너로부터의 등방성 거리로 근사하며, 실제 부착 길이·처짐·미끄럼·접촉 형상을 완전한 케이블/필름 요소로 풀지 않습니다.
- 온도, 점탄성, 주름, 기포, 국부 모서리 형상, stick-slip은 보정 전 정량 예측 대상이 아닙니다.
- 면내 힘과 Mz는 경량 판 모델의 등가 굽힘·비틀림 하중으로 환산됩니다.
- 고정된 등가 인장강성 2.25 N/mm은 실제 자유 필름·풀테이프의 대변형, 처짐, 미끄럼과 점탄성을 하나의 선형 스프링으로 축약한 추정값입니다. 초기장력 조합 민감도는 정보 부족을 드러내지만 실제 장력을 식별하지는 않습니다.
- cohesive 파라미터, 장비 최대 인장력, 초기장력과 유효 자유길이는 로드셀·고속 영상·실제 박리 전선 데이터로 보정해야 합니다. 장비의 실제 테이프 사양을 알게 되면 고정값을 다시 산정해야 합니다.
- 절대 합격 기준을 정하려면 실제 공정의 양품/불량 라벨과 독립 검증 세트가 필요합니다.
- 완성 셀 R, 패드, 아일랜드와 내부 홀 치수는 아직 공정 개념 값이며 실제 도면 치수로 보정해야 합니다.
